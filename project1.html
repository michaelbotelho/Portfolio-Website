Inventory Management System

View source code ↗

Tech Stack:

  • Google Remote Procedure Call Framework
  • Redis Cached Database
  • Python

Key Features:

  • User-Friendly Command Line Interface
  • Add products, Update quantities, Delete products, Fetch all products or Query by ID
  • Fast data Storage and Retrieval with Redis

This application is a distributed Python-based model designed to streamline the management of product inventory. Leveraging gRPC (Google Remote Procedure Call) for communication between client and server, this system allows users to perform various operations such as adding new products, updating quantities, retrieving product details by ID, deleting products, and fetching a list of all available products.

The server side of the system utilizes Redis, an open-source, in-memory data structure store, for efficient data storage and retrieval. The server is configured for multithreading to allow for multiple client connections simultaneously. The client-side interface, designed for ease of use, provides a simple command-line interface for users to interact with the inventory management functionalities seamlessly. With its robust architecture and user-friendly interface, the Inventory Management System could be scaled to offer a reliable solution for maintaining and organizing product inventories effectively.
