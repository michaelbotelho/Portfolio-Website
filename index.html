Product Inventory Manager

View source code ↗

The Inventory Management System is a distributed Python-based application designed to streamline the management of product inventory. Leveraging gRPC (Google Remote Procedure Call) for communication between client and server, this system allows users to perform various operations such as adding new products, updating quantities, retrieving product details by ID, deleting products, and fetching a list of all available products.

The server side of the system utilizes Redis, an open-source, in-memory data structure store, for efficient data storage and retrieval. The server is configured for multithreading to allow for multiple client connections simultaneously. The client-side interface, designed for ease of use, provides a simple command-line interface for users to interact with the inventory management functionalities seamlessly. With its robust architecture and user-friendly interface, the Inventory Management System could be scaled to offer a reliable solution for maintaining and organizing product inventories effectively.
